# YAMAPアウトドア保険 加入者アンケート分析

YAMAPアウトドア保険の加入者アンケートデータを分析し、マーケティング施策に活用するためのインサイトを抽出するプロジェクトです。

## リサーチクエスチョン

1. **属性ごとの加入動機、価値（便益・独自性）、加入タイミング、経路** ⇧主目的
2. **7日プランから年プランへのアップセル経験者のインサイト**
3. **外あそび1年の継続・非継続理由**

## ファイル構成

- `analyze_research_questions.py` - リサーチクエスチョンに基づく詳細分析スクリプト
- `analyze_survey.py` - 基本統計分析スクリプト
- `create_marketing_insights.py` - マーケティング施策向けインサイトレポート作成スクリプト
- `marketing_insights_report.md` - マーケティングインサイトレポート（Markdown）
- `marketing_insights_report.json` - マーケティングインサイトレポート（JSON）
- `yamap_analysis_report.xlsx` - Excel形式の詳細レポート
- `visualizations/` - 分析結果の可視化グラフ

## 使用方法

### 必要なライブラリ

pip install pandas numpy matplotlib seaborn openpyxl

## 主要なインサイト

### 属性別の特徴

- **60代以上**: 「家族への責任」を重視
- **30-40代**: 「手続きの簡単さ」を重視
- **全年代**: 「YAMAPアプリ内バナー」が主要認知経路（約55-65%）

### 加入タイミング

- 約60%が「年間を通した補償を検討」で加入
- 約30%が「直前・前日」に加入

### アップセル

- 短期→年契約への切り替え率: 約1.7%
- 現在短期プラン加入者の約70%は年契約への切り替えを「あまり/全く検討していない」

### 継続・非継続

- 継続率: 約75%
- 主要継続理由: 1年を通した安心（67.5%）、コスパ（48.9%）、登山頻度の高さ（44.4%）
